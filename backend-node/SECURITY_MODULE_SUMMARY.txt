✨ FLUJO DE LOGIN CON GOOGLE (HU-01):

   1. Frontend: Usuario abre app → click "Login con Google"
   2. Frontend: Google SDK retorna idToken
   3. Frontend: POST /api/auth/login { idToken }

   4. AuthService.login():
      ├─ Valida DTO con Zod ✅
      ├─ Verifica idToken con firebase-admin ✅
      ├─ Busca usuario en Firestore (perfiles_seguridad) ✅
      ├─ Verifica usuario.activo === true ✅
      ├─ Genera JWT interno con permisos ✅
      └─ Retorna { token, permisos, usuario, expiresIn }

   5. Frontend: Guarda token en localStorage
   6. Frontend: Solicitudes posteriores incluyen: Authorization: Bearer JWT

🔄 FLUJO PROTECCIÓN DE RUTAS:

   Middleware requireAuth(token):
   ├─ Extrae token de header
   ├─ authService.verificarToken(token)
   ├─ Valida firma JWT ✅
   ├─ Valida expiración ✅
   └─ Inyecta request.user = ITokenPayload

   Middleware requirePermissions(...permisos):
   ├─ Verifica request.user.permisos
   ├─ Rechaza con 403 si no tiene permisos
   └─ Continúa al controlador
